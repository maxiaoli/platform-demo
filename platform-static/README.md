platform-static
=================

这个模块是作为前后端分离情况下，前端项目源文件存放目录存在的。

在前后端分离情况下，后端只提供数据，并处理提交请求数据处理逻辑，即MVC中的M。

而前端则负责数据渲染以及路由转发，即MVC中的VC。


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
